# Umoja Learning Management System (ULMS)

## Overview
The Umoja Learning Management System (ULMS) is a Windows Forms application developed using C# and .NET. The system was tested and debugged as part of a software testing and quality assurance assessment.

## Features
- User login authentication
- Student registration
- Marks capture
- Average calculation
- Report generation
- Navigation between modules

## Technologies Used
- C#
- .NET Windows Forms
- Microsoft Visual Studio 2026
- GitHub

## Testing Activities Performed
- Functional testing
- Non-functional testing
- Input validation testing
- Security testing
- Performance testing
- Debugging and defect correction

## Defects Corrected
- Incorrect login validation
- Empty input handling
- Numeric validation issues
- Incorrect average calculation
- Duplicate student registration validation

## How to Run the Project
1. Open the solution in Visual Studio
2. Build the solution
3. Run the application using F5
